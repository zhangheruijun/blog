在做下拉框中，子元素内容不固定，高度也无法确认，这种情况下展开与收起的动效不好做，用js可以解决，但是用css更好

用max-height代替height

css
/* 展开时 */
.unfold{
  max-height: 10vh;
  overflow: auto;
  transition: max-height 0.6s ease-in-out;
}

/* 收起时 */
.packup{
  max-height: 0;
  overflow: hidden;
  transition: max-height 0.5s cubic-bezier(0,1,0,1);
}
