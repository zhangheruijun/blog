1.scrollHeight和scrollWidth（不包含滚动条） #

获取元素整个内容的高度和宽度 （包含看不见的） ，如果有滚动条（滚动条会占用部分宽高），不计算滚动条的宽高
属性返回一个整数值（小数会四舍五入），表示当前元素的总高度（单位像素），包括溢出容器、当前不可见的部分。它包括padding，但是 不包括border、margin以及滚动条的宽高高度（如果有滚动条的话），还包括伪元素（::before或::after）的高度。 必须是块级元素
scrollWidth: 真实的宽度(包括溢出的宽度) + padding
scrollHeight: 真实的高度(包括溢出的高度) + padding
scrollWidth与scrollHeight的第一种情况：
有滚动条或者设置了overflow-y: scroll/auto/hidden,使子元素不可见
子元素整体小于等于父元素宽高：父元素(width/height) + 父元素padding
2.子元素整体大于父元素的宽高：子元素(width/height + padding + border + margin) + 父元素padding
scrollWidth与scrollHeight的第二种情况：
未设置：overflow-y: scroll/auto/hidden的情况
1.子元素整体小于等于父元素(宽高 + pading-bottom/left)：父元素(width/height) + 父元素padding
2.元素整体大于等于父元素(宽高 + pading-bottom/left)：子元素(width/height + padding + border + margin) + 父元素padding-top
溢出的内容包括overflow: hidden、auto

2.scrollTop和scrollLeft（可读写） #

scrollLeft: 表示当前元素的水平滚动条向右侧滚动的像素数量
scrollTop: 表示当前元素的垂直滚动条向下滚动的像素数量。对于那些没有滚动条的网页元素，这两个属性总是等于
